Level Data Plotter — เวอร์ชันติดตั้งบนมือถือ (PWA)
======================================================

ไฟล์ในชุดนี้:
  index.html          - ตัวโปรแกรม
  manifest.json        - ข้อมูลแอป (ชื่อ, ไอคอน, สีธีม)
  service-worker.js    - ทำให้เปิดใช้แบบออฟไลน์ได้บางส่วนหลังติดตั้งครั้งแรก
  icon-192.png / icon-512.png - ไอคอนแอป

ทำไมต้อง "โฮสต์" ก่อนติดตั้ง
-----------------------------
เบราว์เซอร์มือถือ (Chrome/Safari) จะอนุญาตให้ "ติดตั้งลงหน้าจอโฮม" แบบเต็มรูปแบบ
(รวม service worker + ใช้งานออฟไลน์) ก็ต่อเมื่อเปิดผ่าน https:// เท่านั้น
เปิดไฟล์ตรง ๆ แบบ file:// บนมือถือจะใช้กราฟ/ฟังก์ชันหลักได้ปกติ แต่ปุ่ม
"ติดตั้งลงมือถือ" และการทำงานออฟไลน์อาจไม่ขึ้นหรือไม่สมบูรณ์

วิธีที่ 1 — ใช้งานเร็วที่สุด (ไม่ต้องติดตั้งจริง)
--------------------------------------------------
เปิดไฟล์ index.html ด้วยเบราว์เซอร์บนมือถือได้เลย ใช้งานกราฟ/OCR/PDF ได้ปกติ
เพียงแต่จะไม่มีไอคอนแอปบนหน้าจอโฮมแบบเต็มรูปแบบ

วิธีที่ 2 — ติดตั้งเป็นแอปจริงบนมือถือ (แนะนำ)
-------------------------------------------------
ต้องอัพโหลดทั้งโฟลเดอร์นี้ขึ้นโฮสติ้งที่รองรับ https:// ก่อน วิธีที่ง่ายและฟรี
ที่สุดคือ GitHub Pages (คล้ายกับที่เคยใช้กับ คัมภีร์ช่างกล PWA):

1. สร้าง repository ใหม่บน GitHub (หรือใช้ repo เดิม)
2. อัพโหลดไฟล์ทั้งหมดในโฟลเดอร์นี้ขึ้นไป (index.html, manifest.json,
   service-worker.js, icon-192.png, icon-512.png)
3. เปิด Settings > Pages เลือก branch ที่อัพโหลดไว้ แล้วกด Save
4. รอสักครู่ จะได้ลิงก์ประมาณ https://<username>.github.io/<repo>/
5. เปิดลิงก์นั้นด้วยมือถือ (Chrome บน Android หรือ Safari บน iPhone)

วิธีติดตั้งจากมือถือหลังเปิดลิงก์แล้ว:
  Android (Chrome):
    - จะมีปุ่ม "📲 ติดตั้งลงมือถือ" ขึ้นเองที่มุมขวาบน กดแล้วยืนยัน
    - หรือกดเมนู 3 จุด > "ติดตั้งแอป" / "เพิ่มไปยังหน้าจอหลัก"
  iPhone/iPad (Safari):
    - กดปุ่ม Share (ไอคอนสี่เหลี่ยมมีลูกศรชี้ขึ้น) ที่แถบด้านล่าง
    - เลือก "เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen) แล้วกด "เพิ่ม"

หลังติดตั้งแล้วจะมีไอคอนแอปสีธีมเดียวกับโปรแกรม (พื้นเข้ม เส้นสีเขียวอมฟ้า
ในกรอบสีส้ม) อยู่บนหน้าจอโฮม เปิดได้เหมือนแอปทั่วไป ไม่มีแถบ URL ของเบราว์เซอร์

หมายเหตุ
--------
- ฟีเจอร์ OCR (ถ่ายรูป/นำเข้ารูป) และ Export PDF ยังต้องใช้อินเทอร์เน็ตตอนโหลด
  ไลบรารีจาก CDN ในการเปิดใช้ครั้งแรกของแต่ละเซสชัน
- ถ้าไม่สะดวกทำ GitHub Pages แจ้งมาได้ จะแนะนำโฮสติ้งฟรีทางเลือกอื่นเพิ่มเติม
  (เช่น Netlify, Cloudflare Pages) ให้ได้เช่นกัน
